# Projeto Flybird :video_game:
- Jogo do tipo FlappyBird para dispositivos Android e IOS

- [Expo](https://expo.dev/)

- [React Native Game Engine](https://github.com/bberak/react-native-game-engine)
- [Matter Js](https://brm.io/matter-js/)
